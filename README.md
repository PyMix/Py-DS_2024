# Setup

**Prerequisites:**

- python >= 3.12

## **Install dependencies:**

### **Using pipenv:**

- from pipfile

```sh
pipenv install
```

- from pipfile.lock (exact versions)

```sh
pipenv sync -d
```

- from requirements.txt

```sh
pipenv install -r requirements.txt
```

### **Using pip:**

```sh
pip install -r requirements.txt
```
